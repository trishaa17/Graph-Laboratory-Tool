# Graph-Laboratory-Tool

Graph-based learning is a fundamental concept in machine learning but often difficult for students to understand and visualise due to its abstract mathematical nature. This project aims to develop an interactive visualization laboratory tool that helps deepen students’ understanding. Instead of passively absorbing information, students can actively construct graphs and observe how structural changes affect mathematical representations and clustering outcomes. AI tools were used to assist in the implementation of the system with a focus on improving conceptual clarity and engagement.
